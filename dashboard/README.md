# 📊 Dark Pattern Detection Dashboard

Full analytics dashboard with Chart.js visualizations for the Dark Pattern Detector project.

## Run
```bash
pip install flask
python app.py
```
Open: http://localhost:5004

## Features
- Real-time stats: total analyses, avg score, high risk count
- Risk distribution donut chart
- Score trend line chart
- Top patterns bar chart
- Recent analyses table
- Live text analyzer with highlighted output
