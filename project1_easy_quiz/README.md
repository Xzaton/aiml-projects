# 🎯 Project 1: Dark Pattern Awareness Quiz App
**Difficulty:** Easy | **Tech:** Python + Flask  
**By:** Piyush Mishra | SAP: 590024892

## What it does
An interactive 10-question quiz that tests knowledge of dark patterns in UX design.
Features: randomized questions, instant feedback, explanations, scoring, and badges.

## Setup & Run
```bash
pip install flask
python app.py
# Open http://localhost:5000
```

## Features
- 10 curated questions on real dark pattern types
- Color-coded correct/wrong answers
- Explanation shown after each answer
- Score card with badge at the end
- Dark themed UI
